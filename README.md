# Habit Tracker

## Описание
Habit Tracker — это приложение для отслеживания привычек, которое поможет вам улучшить свою продуктивность и организовать повседневную жизнь. С помощью нашего трекера вы сможете легко управлять своими привычками и отслеживать прогресс.

## Особенности
- **Календарь**: Удобный календарь для визуализации ваших привычек и отслеживания прогресса.
- **Список привычек**: Легкий в использовании список привычек, который позволяет добавлять, редактировать и удалять привычки.
- **Темы**: Переключение между 4 различными темами для персонализации интерфейса под ваши предпочтения.
